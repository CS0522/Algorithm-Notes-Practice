### DFS (P269)
* 通过递归 (也就是栈) 可以很好地实现 DFS
* 注意 "死胡同" 和 "岔道口"，也就是递归的 `边界条件` 和 `递归表达式`
* "剪枝"，可以减少时间复杂度
* eg: [8_DFS.cpp](./8_DFS.cpp)

### BFS (P274)
* 通过队列，while 循环迭代进行实现
  ```cpp
  void BFS(...) 
  {
      queue<T> q;
      q.push(s);
      while (!q.empty())
      {
          ...
      }
  }
  ```
* 要点
  * 全局定义一些变量
  * 定义结构体(类)
  * 可能需要在结构体(类)中设置一些具有标记属性的性质
  * 访问的条件判断
* eg: [8_BFS_1.cpp](./8_BFS_1.cpp), [8_BFS_2.cpp](./8_BFS_2.cpp)
